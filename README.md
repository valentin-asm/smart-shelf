# SmartShelf: Персональная цифровая библиотека

## О проекте (Питч)
SmartShelf — это веб-платформа для систематизации личной книжной коллекции. Проект решает проблему хаоса в учете прочитанной литературы, позволяя пользователям в одном месте хранить список книг, отслеживать прогресс чтения и сохранять краткие заметки. Цель проекта — создать удобный инструмент для личной продуктивности и развития привычки чтения.

## Стейкхолдеры
* **Пользователь:** Книголюб, который хочет видеть структуру своей библиотеки и прогресс в чтении.
* **Разработчик:** Студент курса программной инженерии, работающий над реализацией проекта по принципам SDLC.
* **Преподаватель:** Эксперт, оценивающий качество инженерного подхода и соответствие учебным требованиям.

## Цель первой итерации (MVP)
Реализация базового прототипа веб-страницы, обеспечивающего визуализацию списка книг с основной атрибутикой (название, автор, статус «прочитано/в процессе»).

## Критерии успеха (Definition of Success)
* Репозиторий настроен и является "единым источником правды".
* README.md содержит актуальное описание целей и плана работ.
* Структура проекта (папки `docs/`, `design/`) заложена для дальнейшего расширения.
* Главная страница содержит семантически верную разметку и корректно отображает список книг.

## План дальнейших шагов
1. **Разработка требований:** Формулирование функциональных (FR) и нефункциональных (NFR) требований, составление Backlog (User Stories).
2. **Семантическая верстка:** Создание структуры HTML-страниц (индексная страница, страница контактов).
3. **Проектирование интерфейса:** Разработка Low-fidelity вайрфреймов и определение базовой палитры стилей.
